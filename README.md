# Defend My Human

`Defend My Human` is a small side-view HTML5 canvas game about a brave dog protecting their favorite person from incoming cats.

## What It Is

- Plain JavaScript with no external libraries
- Single-screen arcade gameplay sized for a `960x540` canvas
- Intro, playing, and game-over states
- Dog movement, jumping, gravity, cat spawning, score tracking, and health

## How To Run

1. Open `index.html` in a browser.
2. Use `Arrow Left` / `Arrow Right` or `A` / `D` to move.
3. Press `Space` to jump.
4. Press `Enter` on the game-over screen to restart.

## Files

- `index.html`: page shell and embedded styles
- `script.js`: game logic, entities, and rendering
